Proposed Changes

     Frontend (React)

     1. Simplify the Export Modal:
       - Change the checkboxes to a single-select (radio buttons or a dropdown) to choose which plot to export.
       - Keep the text input for the Filename.
       - Keep the "Download" and "Cancel" buttons.
     2. Update Export Logic (executeExport):
       - Identify the target element based on the selection:
           - If "Time Series" $\rightarrow$ use timeSeriesRef.current.
         - If "Histograms" $\rightarrow$ use histogramsRef.current.
       - Pass the target element directly to html2canvas without cloning.
       - Trigger the download with the user-specified filename.
     3. UI Feedback:
       - Add a simple isExporting loading state to the Download button to prevent multiple clicks and signal that the
     process is running.

     Critical Files

     - frontend/src/pages/Analysis.tsx: Update state, modal UI, and executeExport function.

     Verification Plan

     1. Single Plot Export:
       - Select "Time Series" $\rightarrow$ Verify only the line chart is downloaded.
       - Select "Histograms" $\rightarrow$ Verify only the bar charts are downloaded.
     2. Filename Customization:
       - Enter a custom filename $\rightarrow$ verify the saved file uses that name.
     3. Loading State:
       - Verify the button shows a loading state during the capture process.
     4. Visual Quality:
       - Check that the resulting PNG has the correct background and high resolution.
