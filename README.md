# Road Atlanta Telemetry Mapping Utility

This Python utility processes GPS telemetry data from CSV files to generate a projected 2D track map of Road Atlanta.

## Functionality
* **GPS Projection:** Converts raw data into Cartesian coordinates.
* **Manual Turn Annotation:** Allows for precise x, y placement of turn labels.

## How to Run
1. Install dependencies: pip install pandas numpy matplotlib adjustText
2. Run the script: python manualTurnMarking.py
