Plan: Add Race Track Plot to Analysis Page

 Context

 The goal is to add a visualization of the race track on the Analysis page. This involves projecting GPS coordinates
 (Latitude/Longitude) onto a 2D plane using an equirectangular mapping and rendering the resulting X,Y plot. This
 allows users to see the physical path of the vehicle.

 Implementation Approach

 1. GPS Projection Utility

 Create a new utility file frontend/src/utils/gpsUtils.ts to implement the math from manualTurnMarking.py.
 - Constants: Earth Radius $R = 6,371,000$ meters.
 - Logic:
   - Convert degrees to radians.
   - $x = R \cdot (\text{lon}{rad} - \text{refLon}{rad}) \cdot \cos(\text{refLat}_{rad})$
   - $y = R \cdot (\text{lat}{rad} - \text{refLat}{rad})$
 - Function: projectGPS(lat, lon, refLat, refLon) returning { x, y }.

 2. TrackPlot Component

 Create a new component frontend/src/components/TrackPlot.tsx to handle the visualization.
 - Data Fetching: The component will fetch data independently from /files/data using fileId and explicitly requesting
 GPS Latitude and GPS Longitude.
 - Data Cleaning:
   - Zero Removal: Filter out any points where latitude or longitude are $0$ (signal loss).
   - Reference Point: The first valid (non-zero) point will be used as the reference origin ($\text{lat}_0,
 \text{lon}_0$).
   - Outlier Removal: Implement a filter to remove GPS spikes/outliers (e.g., based on distance from previous point or
 simple IQR) to prevent graph distortion.
 - Projection: It will map all remaining points using projectGPS.
 - Visualization: Use recharts ScatterChart.
   - Points will be small to create a continuous line.
   - Axes will be labeled "Meters (X)" and "Meters (Y)".
   - A custom tooltip will show original GPS coordinates.
 - Styling: Follow the "Apple Liquid Glass" aesthetic (translucent backgrounds, subtle borders).

 3. Analysis Page Integration

 Modify frontend/src/pages/Analysis.tsx:
 - State: Add 'trackPlot' to the activeTab state union.
 - Activation Logic: Create a derived boolean hasGPSColumns that checks if availableColumns contains both 'GPS
 Latitude' and 'GPS Longitude'.
 - UI:
   - Add a "Track Plot" button to the button row (next to "Check Diagnostics").
   - Button enabled if selectedFile exists and hasGPSColumns is true.
